THE CSS THAT MAKES YOU FEEL GOOD.

Fluid typography, spacing, and grid layouts — all generated from class names. Zero runtime. Pure CSS output.

v0.1.0

Quick Start

npm install
npm link              # registers `dopamine` globally
npm run build         # scan HTML → generate SCSS → compile CSS
npm run dev           # watch + Sass + browser auto-reload
