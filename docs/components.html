Accordion

CSS-only accordion using native <details> / <summary> elements. No JavaScript. Styled with dp.fluid() in a separate component file: scss/components/accordion.scss

What is Dopamine Fluid?
A CSS utility generator that creates fluid clamp() values and grid layouts from class names in your HTML. No runtime, no JavaScript — just scan your templates and generate pure CSS.
How does the Sass function work?
Import _dopamine-functions.scss and use dp.fluid(16, 48) anywhere in your Sass. Same clamp() math as the utility classes, for elements where you can't add class names.
Can I use it with Drupal?
Yes — scan your Twig templates, generate SCSS, and compile alongside your theme's existing Sass pipeline. Each component gets its own CSS file for Drupal's library system.
What browsers are supported?
All modern browsers. This accordion uses native <details>/<summary> elements — works in Chrome, Firefox, Safari, and Edge.

Sass Integration dp.fluid()

When Dopamine outputs to .scss, it generates a _dopamine-functions.scss file with a fluid() Sass function. Use it for elements where you can't add utility classes (e.g. Drupal-rendered content).

Import and use

@use 'dopamine-functions' as dp;

// Elements you can't add classes to
.node--article .field--body p {
  font-size: dp.fluid(16, 48);
  margin-bottom: dp.fluid(8, 24);
}

// Custom viewport override
.hero-banner h1 {
  font-size: dp.fluid(32, 96, 480, 1920);
}

Function signature

@function fluid($min, $max, $vpMin: 320, $vpMax: 1440)
// $min    — minimum value in pixels
// $max    — maximum value in pixels
// $vpMin  — viewport min (defaults from config)
// $vpMax  — viewport max (defaults from config)
// Returns: clamp(minRem, calc(Xvw + Yrem), maxRem)

Same math as the utility classes. Same viewport defaults from your dopamine.config.json.

Component File Structure

Each .scss file in scss/components/ compiles to its own CSS file. This means you can load them per-page in Drupal's library system.

File layout

scss/components/
  accordion.scss     → css/components/accordion.css
  hero.scss          → css/components/hero.css
  slider.scss        → css/components/slider.css

Example component

// scss/components/accordion.scss
@use '../dopamine-functions' as dp;

.accordion__title {
  font-size: dp.fluid(18, 24);
  padding: dp.fluid(12, 24);
  font-weight: 600;
  background: #1a1a2e;
  color: #fff;
}

Drupal libraries.yml

global-styling:
  css:
    theme:
      css/main.css: {}

accordion:
  css:
    component:
      css/components/accordion.css: {}
